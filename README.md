## 简介

AI 写作工作台是一个可放入 U盘随身携带的 AI 写作软件。覆盖 8 个行业、58 个专业模板，**无需安装任何软件**，双击即用。

内置完整 Python 运行环境，不依赖用户电脑安装任何东西。支持 DeepSeek、Green-API、Ollama 本地和自定义 API 四种 AI 引擎。

## 特性

- 🎯 **8 个行业版本**：党政、商务、法务、教师、学生、职场、媒体、通用
- 📋 **写作向导**：选模板 → 填字段 → AI 智能生成，无需学习 AI 指令
- 💬 **AI 对话**：随时与 AI 交流写作内容
- 🎨 **8 套主题色**：切换版本自动换肤
- 📤 **多格式导出**：Markdown / HTML / Word / PDF / 微信公众号（5 种排版风格）
- 🔍 **模型搜索**：Green-API 和自定义 API 支持一键获取可用模型列表
- 🔄 **系统升级**：页面内一键检查 GitHub 更新
- 🐍 **内置 Python**：`_env/` 60MB 完整环境，零依赖
- 🔒 **隐私安全**：文档保存在 U盘本地，不上传云端

## 快速开始

### 第一步：填写 API 密钥

用记事本打开 U盘里的 `.env.local` 文件，填入 DeepSeek API Key：

```ini
DEEPSEEK_API_KEY=sk-你的密钥
```

> 💡 免费注册：https://platform.deepseek.com

### 第二步：启动

双击 **`启动写作台.bat`**

首次启动会自动下载 Python 运行环境（约 60MB，仅首次需要网络）。

### 第三步：开始写作

1. 浏览器自动打开首页，点击想用的版本卡片（如「党政写作」）
2. 左侧「写作向导」选择模板（如「通知」）
3. 填写字段：发文机关、事由、通知内容
4. 点击「🤖 智能生成」
5. AI 自动生成完整公文

每个模板设计了引导式表单，你只需按照字段提示填写信息，无需学习复杂的 AI 指令。

```
示例：党政版写「通知」

1. 选择模板 → 通知
2. 填写字段：
   - 发文机关：XX局办公室
   - 主送机关：各科室
   - 通知事由：关于国庆假期值班安排
   - 通知事项：1.放假安排 2.值班要求 3.安全注意事项
   - 执行要求：请于9月28日前上报值班表
3. 展开「💡 参考范例」查看范文格式
4. 点击「🤖 智能生成」
5. AI 生成完整公文 → 编辑区微调 → 💾 保存
```

## 常见问题

**Q：启动时出现乱码？**

A：不影响使用。如需查看详细信息，在项目目录下打开 PowerShell 运行 `python -B launcher.py`。

**Q：提示 Python 未找到？**

A：首次启动自动下载 Python 环境。请确保此时有网络连接（仅首次需要）。

**Q：AI 回复"请求失败"？**

A：检查 `.env.local` 里的 API Key 是否正确，或 API 账户余额是否充足。打开设置页面重新配置。

**Q：能离线用吗？**

A：可以。安装 Ollama 并下载本地模型（如 Qwen2.5），在设置中切换为 Ollama 引擎即可完全离线。

**Q：文档存在哪？**

A：U盘的 `_data/documents/` 目录下。换台电脑插上 U盘，文档不会丢失。

**Q：能自己加模板吗？**

A：可以。在对应版本的 `templates/` 目录下新建 JSON 文件，参照已有模板的 `fields` 格式编写。

**Q：Word/PDF 导出报错？**

A：以管理员身份运行 `install_libs.bat` 一次安装依赖库，之后正常导出。
