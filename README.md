# AI 写作工作台 v1.0.0

> 🚀 **插上 U盘，双击启动，零门槛的专业 AI 写作助手**

## 这是什么

一个可以放在 U盘里随身携带的 AI 写作软件。覆盖 8 个行业、57 个专业模板，无需安装任何软件，插入电脑双击即用。

## 电脑要求

- Windows 10 或更高版本
- 建议 4GB 以上内存
- 无需安装 Python 或其他软件
- U盘容量 64GB 以上（项目占用约 80MB）

## 快速开始（3 步）

### 第一步：填写 API 密钥

用记事本打开 U盘里的 `.env.local` 文件，找到这一行填入你的 DeepSeek API Key：

```
DEEPSEEK_API_KEY=sk-你的密钥填这里
```

> 💡 免费注册获取 Key：访问 https://platform.deepseek.com

### 第二步：启动写作台

双击 U盘里的「**启动写作台.bat**」

### 第三步：开始写作

1. 浏览器自动打开首页
2. 点击你想用的版本卡片（如「党政写作」）
3. 左侧「写作向导」选择模板
4. 填写关键信息
5. 点击「🤖 智能生成」

## 详细使用指南

### 写作向导

每个模板都设计了引导式表单，你只需要按照字段提示填写信息，无需学习复杂的 AI 指令。

```
举例：使用党政版写一篇通知

1. 点击首页「🏛 党政写作」
2. 左侧选模板 →「通知」
3. 按字段填写：
   - 发文机关：XX局办公室
   - 主送机关：各科室
   - 通知事由：关于国庆假期值班安排
   - 通知事项：放假日程 / 值班要求 / 安全注意事项
   - 执行要求：请各科室于9月28日前上报值班表
4. 点击「🤖 智能生成」
5. AI 生成完整公文 → 编辑区修改 → 💾 保存
```

**微信公众号导出**支持 5 种风格，点击后右侧实时预览：

| 风格 | 适用场景 |

| 💼 极简商务风 | 企业号、行业分析、职场干货 |

## 常见问题

**Q：启动时出现乱码？**
A：不影响使用。如果想知道具体信息，在项目目录下打开 PowerShell，运行 `python launcher.py` 查看。

**Q：提示 Python 未找到？**
A：首次启动会自动下载 Python 环境。请确保此时有网络连接，下载完成后以后都无需网络。

**Q：AI 回复 "API 请求失败"？**
A：检查 `.env.local` 里的 API Key 是否正确填写，或者 API 账户余额是否充足。

**Q：能离线使用吗？**
A：需要安装 Ollama 并下载本地模型（如 Qwen2.5），然后在设置里切换为 Ollama 引擎即可完全离线。

**Q：文档保存在哪里？**
A：在 U盘的 `_data/documents/` 目录下。换电脑插入 U盘，文档不会丢失。

**Q：可以自己加模板吗？**
A：可以。在对应版本的 `templates/` 目录下新建 JSON 文件，按已有模板的格式写即可。

**Q：为什么导出 Word 或 PDF 会报错？**
A：需要以管理员身份运行一次 `install_libs.bat` 安装依赖库，之后即可正常导出。

**Q：写作台能同时运行多个版本吗？**
A：一个写作台中可以在不同版本之间自由切换。点击顶栏「← 首页」返回选择界面，再点击其他版本即可。
