## 简介

一个可放入 U盘随身携带的 AI 写作软件，覆盖 8 个行业、58 个专业模板，**无需安装任何软件**，双击即用。

内置完整 Python 运行环境 + **离线本地大模型支持**（llama.cpp），插上 U盘就能离线写作，隐私数据不出电脑。

## 特性

- 🎯 **8 个行业版本**：党政、商务、法务、教师、学生、职场、媒体、通用
- 📋 **写作向导**：选模板 → 填字段 → AI 智能生成
- 💬 **AI 对话**：随时与 AI 交流写作内容
- 🧠 **离线大模型**：内置 llama.cpp + 千问模型，完全离线
- 📤 **多格式导出**：MD / HTML / Word / PDF / 公众号（5 种风格）
- 🎨 **8 套主题色**：切换版本自动换肤
- 🔄 **系统升级**：GitHub 一键升级
- 🔒 **隐私安全**：文档保存在 U盘，不上传云端

## 快速开始

### 第 1 步：启动

双击 **`启动写作台.bat`**，浏览器自动打开。

### 第 2 步：选择引擎

| 引擎 | 说明 | 需要网络 |
|------|------|---------|
| DeepSeek | 云端，中文首选 | ✅ |
| Green-API | API 中转 | ✅ |
| 本地模型 | U盘内置千问 | ❌ |
| 自定义 API | 自由配置 | ✅ |

### 第 3 步：开始写作

点击版本卡片 → 左侧写作向导选模板 → 填写信息 → 点生成。

### 云端引擎

| DeepSeek | https://api.deepseek.com | 可填写模型名（deepseek-chat / deepseek-reasoner） |

| 内存 | 推荐模型 | 大小 |
|------|---------|------|
| 8GB | Qwen2.5-1.5B | ~1.0 GB |


## 常见问题

**Q：启动乱码？** A：不影响使用。可运行 `python -B launcher.py` 查看详情。

**Q：AI 请求失败？** A：检查 API Key 是否正确或余额是否充足。

**Q：能离线用吗？** A：可以。下载千问模型放入 `_models/`，切换到本地引擎即可。

**Q：文档存哪？** A：`_data/documents/`，换电脑不丢失。

**Q：能自己加模板？** A：在 `templates/` 下新建 JSON，参照已有格式。

**Q：导出 Word/PDF 时 360 报毒？** A：安全软件误报。将写作台文件夹加入 360 白名单：设置 → 文件白名单 → 添加 `xiezuoUpan` 目录。

**Q：360 拦截 llama-server.exe？** A：360 USB 防护策略：U盘里的 .exe 一律拦截，不是病毒。将 `_models\llama-server.exe` 加入 360 信任区即可。

**Q：如需发布新版本？** A：运行 `python release.py` 自动更新版本号并生成 zip，然后 `git push --tags`。
